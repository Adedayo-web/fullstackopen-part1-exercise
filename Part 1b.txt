Variables
------------
	These are few ways of defining variables:
const x = 1;
let y = 5;

console.log(x, y);
y += 10;
console.log(x, y);
y = "sometext";
console.log(x, y);
x = 4;

const is used to define a variable whose value cannot be changed. let is used to define a normal variable whose value can change over time. it is also possible to define variables in JS using var.


Arrays
--------
	An array allows users to store a collection of multiple items under a single variable name and has members for performing common array operations.

const t = [1, -1, 3];

t.push(5);

console.log(t.length);
console.log(t[1]);

t.forEach((value) => {
  console.log(value);
});

in the code above we can push 5 to the array because const prevents users from reassigning a variable (e.g. saying y = 1) but it doesn't prevent users from modifying in the array.

push is an array operation used to add values to an array. 
pop is an operation that only removes the last element in an array (t.pop()). 
forEach receives a function defined using the arrow syntax as a parameter, it calls the function for each of the items in the array. 

const t = [1, -1, 3];
const t3 = [2, 4, 5];
const t2 = t.concat(t3);

console.log(t);
console.log(t2);

t.forEach((value) => {
  console.log(value);
});

const t6 = [1, -1, 3];
const t7 = 2
const t8 = t6.concat(t7);

console.log(t6);
console.log(t8);

concat() is used to merge two or more arrays or variables together . it doesn't add items or merge the arrays it just creates a new array.

const t = [1, -1, 3];

const m1 = t.map((value) => value - 2);
console.log(m1);

map() lets user change the items in an array based on the conditions the user gave and creates a new array returning the changed items. the condition in the code above is (value - 2).

other ways of using map()

const t = [1, -1, 3];

const m1 = t.map((value) => "<li>" + value + "<li>");
console.log(m1);


destructuring assignment is a JavaScript syntax that allows users to unpack values from arrays into distinct variables.

const t = [1, 2, 3, 4, 5];

const [first, second, ...rest] = t;

console.log(first, second);
console.log(rest);
console.log(t);

the first and second represent the first and second value of the array and they can be called anything you want. the rest can also be called anything of choice e.g. 
const [y, x, ...z] = t;

rest is used to put all the remaining elements in the array into one array.


Objects
--------
	One of the common ways of defining an object is using object literals.

const object1 = {
  name: "Arto Hellas",
  age: 35,
  education: "PhD",
};

const object3 = {
  name: { first: "Tosin", last: "Balogun" },
  grade: [2, 3, 4, 5],
  department: "Stanford University",
};

the above method is the method of creating using Objects literals. the values are stored in properties and the values can be of any type like strings, arrays, objects etc. To reference the properties of an object we do that using dot notation like in the example below:

const object1 = {
  name: "Arto Hellas",
  age: 35,
  education: "PhD",
};

const object3 = {
  name: { first: "Tosin", last: "Balogun" },
  grade: [2, 3, 4, 5],
  department: "Stanford University",
};
const object2 = {
  name: "Full Stack",
  level: "intermediate Studies",
  size: 5,
};

console.log(object3.grade[3]);
console.log(object2.name); /* we use dot notation when we are calling an object property directly */

const fieldName = "age";
console.log(object1[fieldName]); /* we used bracket notation because we are calling a variable */


We can also add properties to an object using dot or bracket notation. e.g.

const object3 = {
  name: { first: "Tosin", last: "Balogun" },
  grade: [2, 3, 4, 5],
  department: "Stanford University",
};

object3.address = "13th Avenue";
object3["Phone Number"] = +9123456789;

console.log(object3);

note:
"Phone Number" works but depart ment doesn't because phone number has double quotes and depart ment doesn't  make JS think depart and ment are two different words. so to make it work do this instead

const object3 = {
  name: { first: "Tosin", last: "Balogun" },
  grade: [2, 3, 4, 5],
  "depart ment": "Stanford University",
};

console.log(object3["depart ment"]);



Functions
==========
using parameters on an arrow function

for an arrow function with two parameters 
const sum = (p1, p2) => {
  console.log(p1);
  console.log(p2);
  return p1 + p2;
};

const result = sum(1, 5);
console.log(result);

sum(3, 5); // I just want to return the function directly.
console.log(sum(3,5)) // to return what was added.


for single parameter arrow function
const mul = n => { /* n doesn't need a bracket since it is only one parameter but it can be added (n) */

  return n * n;
};

if the single parameter contains one expression we can do

const div = (m) => m / m;
console.log(div(5));

the one line arrow function is particularly useful when manipulating arrays. e.g.

const t = [1, 2, 3];
const tSquared = t.map((p) => p * p);
console.log(tSquared);


REGULAR FUNCTION
a regular function with two parameters
function product(a, b) {
  return a + b;
}
console.log(product(1, 3));

another way of declaring a regular function is using

const average = function (a, b) {
  return (a + b) / 2;
